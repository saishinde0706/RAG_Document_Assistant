# RAG_Document_Assistant
AI_RAG_Assistant

Watsonx PDF Q&A Bot is an AI-powered document assistant that lets you ask questions about any PDF you upload and get accurate, context-aware answers instantly.
Built using Python, IBM watsonx.ai, LangChain, and Gradio, it provides a clean and interactive interface to explore and query your documents.
