********RAG Document Assistant** is an AI-powered document assistant that lets you ask questions about any PDF you upload and get context-aware answers instantly.
Built using **Python**, **IBM watsonx.ai**, **LangChain**, and **Gradio**, it provides a clean and interactive interface to make your documents searchable and understandable.

---

##Features

- **AI Question Answering** — Uses IBM watsonx.ai models to read and answer questions from PDFs.
- **PDF Upload Interface** — Simple drag-and-drop PDF upload via Gradio.
- **Easy Setup** — Quick to install and run locally.
- **Customizable Queries** — Ask anything about your document; supports detailed and contextual answers.

---

##Tech Stack

- **Python 3.8+**
- **IBM watsonx.ai (LLM + Embeddings)**
- **LangChain (RAG pipeline and text splitting)**
- **Chroma (vector database for embeddings)**
- **Gradio (interactive web interface)**
- **PyPDFLoader (PDF document loader)**
