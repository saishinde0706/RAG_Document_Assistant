# RAG_Document_Assistant
AI_RAG_Assistant

Watsonx PDF Q&A Bot is an AI-powered document assistant that lets you ask questions about any PDF you upload and get accurate, context-aware answers instantly.
Built using Python, IBM watsonx.ai, LangChain, and Gradio, it provides a clean and interactive interface to explore and query your documents.

Features:

AI Question Answering — Uses IBM watsonx.ai models to understand and retrieve answers from your PDF content.

PDF Upload Interface — Upload your PDF and instantly query it via a simple Gradio web app.

Easy Setup — Lightweight Python project, easy to install and run locally.

Customizable Queries — Ask anything about your document; supports complex and contextual questions.

Vector Search — Uses embeddings and a vector database (Chroma) to retrieve relevant information efficiently.

Tech Stack:

Python 3.8+

IBM watsonx.ai (LLM + Embeddings)

LangChain (for RAG pipeline and text splitting)

Chroma (vector database for embeddings)

Gradio (for interactive web interface)

PyPDFLoader (PDF document loader)
